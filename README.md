## 🚀 Featured Project: BinTrio

**BinTrio** is a gamified smart waste-management Android app that connects Firebase, Android, and ESP32-based smart dustbin hardware.

### Key Highlights

- Android app with modern UI
- Firebase Realtime Database integration
- Score, rank, and badge system
- Daily, weekly, and monthly leaderboard
- Achievement and chest reward system
- ESP32 smart dustbin integration
- Future plan: ESP32-CAM face recognition and waste verification
