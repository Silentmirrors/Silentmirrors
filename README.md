## 🗑️ Featured Project: BinTrio

**BinTrio** is a gamified smart waste-management app that turns cleanliness into a rewarding experience.

### Key Highlights

- Android app with modern UI
- Firebase Realtime Database integration
- User score and rank system
- Daily, weekly, and monthly leaderboard
- Achievement and chest reward system
- ESP32 smart dustbin hardware concept
- Waste detection using sensors
- Future plan: ESP32-CAM face recognition and camera-based waste verification
